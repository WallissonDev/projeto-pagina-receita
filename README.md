## 📋 Sobre o Projeto

Página de receita culinária apresentando o passo a passo do **Cupcake de Café com Chantilly**, com imagem de destaque, lista de ingredientes e modo de preparo completo. O projeto teve foco em aplicar **HTML semântico** com uso correto das tags estruturais, combinado com uma estilização limpa e agradável em CSS.

---

## ✨ Funcionalidades

- ✅ Imagem de destaque do prato
- ✅ Seção de apresentação com tempo, rendimento e dificuldade
- ✅ Lista de ingredientes organizada
- ✅ Modo de preparo em texto corrido
- ✅ Tipografia elegante com fonte **Alice** (Google Fonts)

## 🧠 Foco Técnico — HTML Semântico

Este projeto foi pensado para consolidar o uso correto das tags semânticas do HTML5. Alguns conceitos aplicados:

- `<main>` para delimitar o conteúdo principal da página
- `<section>` para separar as áreas de apresentação, ingredientes e preparo
- `<footer>` para o rodapé com créditos
- `<h1>` e `<h2>` respeitando a hierarquia de títulos
- `<ul>` e `<li>` para a listagem semântica de ingredientes
- `<img>` com atributo `alt` descritivo para acessibilidade
